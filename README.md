# still-images
Still images

## Description
This project is a simple webpage that recreates a **minimal still-life scene using HTML and CSS**.
The composition includes a table surface, an iPad, and an orange placed on top of the table.

The goal of the project is to practice **basic layout, positioning, and CSS styling**
such as:
 - Absolute positioning
 - CSS shapes
 - Borders and shadows
 - Gradients

## Features
 - Table created using a full-width div
 - iPad created using a rectangle shape
 - Orange created using a circular div with gradient
 - Drop shadows to create depth

---

## Example Code
#orange { 
  position: absolute; 
  top: 220px; 
  left: 470px; 
  
  width: 70px; 
  height: 70px; 
  border-radius: 50%; 
  
  background: linear-gradient( 
    to left, 
    orange, 
    transparent 
  ); 
}
